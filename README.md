# ⚙️ Dotfiles

Configurações pessoais (*dotfiles*) para programas que utilizo no desenvolvimento.

## 📦 Programas incluídos

- **Neovim**
- **Tmux**
- **Kitty**

## 🚀 Instalação

Copie o conteúdo de cada pasta para o diretório `.config` do seu usuário.

### Exemplo com Neovim

```bash
cp -r ./nvim/* ~/.config/nvim
